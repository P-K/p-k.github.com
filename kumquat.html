Kumquat

For a course in computer security, my partner and I explored the feasibility of picture passwords. We stated that the human was the weakest link in the security chain (e.g. would pick the weakest password, would be lazy) so we prototyped a picture password that would allow a user to pick a password with greater entropy without much effort.
